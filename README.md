# CLCNER_model
### Requirements

- python 3.9
- pytorch 1.9.0+cu111
- transformers 4.10.0
- CLCNER dataset

Other pip package show in `requirements.txt`.

```bash
pip3 install -r requirements.txt
```


### Train and Evaluate

```bash
bash run.sh
```


## Datasets

We use the CLCNER dataset labeled by ourselves
